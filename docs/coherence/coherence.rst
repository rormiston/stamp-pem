++++++++++++++++++++
Coherence Utilities
++++++++++++++++++++

:py:mod:`stamp_pem.coherence_functions` contains a set of useful functions for calculating coherence. :class:`stamp_pem.coherence_segment.PEMCoherenceSegment` is a class for handling coherence from a single segment of data using :py:mod:`stamp_pem.coherence_functions`. Lastly, :class:`stamp_pem.coherence_segment.PEMSubsystem` is a class for handling the coherence for all of the channels in a given subsystem.
